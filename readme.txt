A WooCommerce plugin that allow user to add a donation or tips at the checkout page.

== Description ==

A WooCommerce plugin that allow user to add customer donation/tip amount option for WooCommerce checkout.

In WooCommerce sidemenu, there will be a new submenu link called 'TT Donation Checkout' where you can:

*   Enabled / Disabled the option
*   Change "Message" to any text
*   Change "Button Label" to any text
*   Change "Title" to any text
*   Change "Default Amount" to any amount
*   Choose "Taxable"
*   Change "Minimum Donation/Tip Amount" to any amount

= Features =

*   Implement an donation/tip checkout for any order
*   2 languages available : English US (en_US) and Chinese (zh_CN)

== Installation ==

1. Upload the entire *woocommerce-donation-checkout* folder to the */wp-content/plugins/* directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'WooCommerce' at left sidebar menu and update the options at Tab *TT Donation Checkout* there.
4. That's it. You're ready to go and cheers!
